Basic SQL

数据库语言可分为两个部分，DDL和DML。

  • DDL（Data Definitoin Language）用于描述数据库中要存储的现实世界实体，操作数据库的结构等
  • DML（Data Manipulation Language）用于数据库操作，操作数据库存储的对象或数据

SQL DDL:数据定义

数据库相关

  • 查询所有数据库
  • 创建新数据库
  • 删除数据库

    表相关

  • 查询当前数据库下所有表
  • 创建新表
  • 删除表
  • 修改已有表的列

    SQL DML:数据操作

  • 从表中获取数据
  • 修改表中数据
  • 删除表中数据
  • 向表中插入数据
  • 去重后从表中获取某列中值
  • 根据条件获取表中数据
  • 根据指定的列对结果集进行排序
